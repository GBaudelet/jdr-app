# FICHE DE PERSO

## OBJECTIF

- permettre aux utilisateurs de créer, personnaliser, modifier, enregistrer et imprimer des fiches de personnage
- Fonctionnalité :
  création de fiche
  ajout de bloc(éléments de création de la fiche)
  gestion des utilisateur
- technologie :
  express
  mysql3
  dotenv
  bcrypt.js
  nodemon
  react
